#1. DMPHub Element Set

The 21 elements listed in Table 1 are derived from our "Rationale of Highlighted Areas" in Step 2 of Section 2: Analysis of Objects, Users, and Use Cases in Description of the Domain. In some cases, we refined the name of the element for clarity, and therefore several elements differ in name from those potential elements described in the aforementioned referenced section.

Collectively, these elements capture what we have found in the literature as well as in our professional and educational experiences in RDM and creating DMPs to be important elements of these plans. These final elements were also named and developed with our potential users and use cases in mind, using an ideation process to understand how users might customize their DMP search based on their use cases.
