Controlled Vocabulary for Analysis/softwarePackage

Twenty software packages and programming libraries most commonly found in data management plans across several disciplines of research. More terms will be added to the controlled vocabulary as the DMPHub project expands

Note: all definitions courtesy of Wikipedia.com

#1

Term Name: SPSS
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/spss
Label Statistical Package for Social Sciences (SPSS)
Definition Software package used for interactive, or batched, statistical analysis.
Comment Current version named IBM SPSS Statistics.

#2

Term Name: R
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/r
Label R
Definition Programming language and free software environment for statistical computing and graphics.
Comment This term includes the use of RStudio.

#3

Term Name: MATLAB
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/matlab
Label MATLAB
Definition Multi-paradigm numerical computing environment and proprietary programming language for matrix manipulations, plotting of functions and data, implementation of algorithms, creation of user interfaces, and interfacing with programs written in other languages.
Comment This term includes the use of MATLAB Online.

#4

Term Name: ArcGIS
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/arcgis
Label ArcGIS from Environmental Systems Research Institute (ESRI)
Definition Geographic information system for working with maps and geographic information, maintained by Esri.
Comment User should use refined vocabulary terms unless using all ArcGIS products in the research, at which point they should use "ArcGIS"
Refines ArcMap; ArcCatalog; ArcGlobe; ArcScene

#5

Term Name: QGIS
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/qgis
Label QGIS
Definition Free and open-source cross-platform desktop geographic information system application that supports viewing, editing, and analysis of geospatial data.
Comment Formerly known as Quantum GIS

#6

Term Name: MAXQDA
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/maxqda
Label MAXQDA
Definition Software program designed for computer-assisted qualitative and mixed methods data, text and multimedia analysis.
Comment Users should note other software used to import data into MAXQDA.

#7

Term Name: Stata
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/stata
Label Stata
Definition General-purpose statistical software package used mainly in the fields of economics, sociology, political science, biomedicine and epidemiology.
Comment Small Stata, which was the smaller, student version for educational purchase only, is no longer available.

#8

Term Name: SAS
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/sas
Label Statistical Analysis System (SAS)
Definition Statistical software suite for advanced analytics, multivariate analysis, business intelligence, criminal investigation, data management, and predictive analytics.
Comment Term includes all subcomponents of the SAS suite. Future iterations of this controlled vocabulary will include refinements for all 200 subcomponents.

#9

Term Name: NVivo
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/nvivo
Label NVivo
Definition Qualitative data analysis computer software package designed for qualitative researchers working with text-based and/or multimedia information.
Comment Users should note other software used to import data into NVivo.

#10

Term Name: ATLAS.ti
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/atlasti
Label ATLAS.ti
Definition Computer program used mainly in qualitative research or qualitative data analysis.
Comment Term includes desktop and cloud version of ATLAS.ti.

#11

Term Name: TensorFlow
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/tensorflow
Label TensorFlow
Definition Free and open-source software library for dataflow and differentiable programming across a range of tasks. It is a symbolic math library, and is also used for machine learning applications such as neural networks.
Comment Term includes all subapplications of TensorFlow, including TensorFlow Lite.

#12

Term Name: Shogun
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/shogun
Label Shogun
Definition Free, open-source machine learning software library written in C++.
Comment Term includes all components of the Shogun toolbox.

#13

Term Name: Minitab
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/minitab
Label Minitab
Definition General-purpose statistical software package designed for easy interactive use.
Comment Term includes all Minitab products, including Minitab 19 and SPM 8.

#14

Term Name: RapidMiner
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/rapidminer
Label RapidMiner
Definition Data science software platform that provides an integrated environment for data preparation, machine learning, deep learning, text mining, and predictive analytics.
Comment Term includes all subproducts of RapidMiner. Future iterations of this controlled vocabulary will include refinements for all subproducts.

#15

Term Name: SciPy
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/scipy
Label SciPy
Definition Free and open-source Python library used for scientific computing and technical computing. Contains modules for optimization, linear algebra, integration, interpolation, special functions, FFT, signal and image processing, ODE solvers and other tasks common in science and engineering.
Comment User should also list any associated software platforms used to run SciPy.

#16

Term Name: OpenRefine
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/openrefine
Label OpenRefine
Definition Standalone open source desktop application for data cleanup and transformation to other formats, the activity known as data wrangling. It is similar to spreadsheet applications; however, it behaves more like a database.
Comment Formerly known as Google Refine.

#17

Term Name: QDAMiner
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/qdaminer
Label QDA Miner
Definition Mixed methods and qualitative data analysis software designed to assist researchers in managing, coding and analyzing qualitative data.
Comment Term includes full version of QDA Miner and Lite version.

#18

Term Name: Oxygen
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/oxygen
Label Oxygen XML Editor
Definition Multi-platform XML editor, XSLT/XQuery debugger and profiler with Unicode support. It is a Java application, so it can run in Windows, Mac OS X, and Linux.
Comment Term includes Oxygen XML Editor, Developer, and Author. Future iterations of this controlled vocabulary will include refinements for these specific subproducts.

#19

Term Name: MicrosoftOffice
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/microsoftoffice
Label Microsoft Office Suite
Definition Suite of products including Microsoft Word, Excel, Access, Publisher, PowerPoint, and Outlook. Each program serves a different purpose and is compatible with other programs included in the package.
Comment User should use the refined vocabulary terms when using specific products, unless the entire suite is being used in the research project, at which point "MicrosoftOffice" should be used.
Refines Word; Excel; Access

#20

Term Name: NeuralDesigner
URI http://www.utkmetadata.org/specification/utk_namespace/analysis/vocabulary/neuraldesigner
Label Neural Designer
Definition Software tool for data analytics based on neural networks, a main area of artificial intelligence research, and contains a graphical user interface which simplifies data entry and interpretation of results.
Comment Term includes both cloud and desktop version of Neural Designer.
